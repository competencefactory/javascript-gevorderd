De Fetch API is een browser API en levert in een interface om resources te fetchen over een netwerk.
De `fetch()` methode returned een `Promise`, welke resolved naar een `Response` op het betreffende request.
Dit gebeurt zowel wanneer het request succesvol is (statuscode 2xx) en in geval van een error (statuscode 4xx).

We maken voor deze oefening gebruik van de [Pok√©mon API](https://pokeapi.co/).

De basis url voor de API call (zonder parameters) is `https://pokeapi.co/api/v2/`

Deze API returned JSON, welke geparsed moet worden. Zie [hier](https://developer.mozilla.org/en-US/docs/Web/API/Response/json) voor meer informatie over de `.json()` methode.
